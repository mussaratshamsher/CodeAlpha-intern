# CodeAlpha-MusicPlayPlayer

This is a simple music player built with HTML, CSS, and JavaScript. It allows you to play, pause, skip, and control the volume of your music. It also includes a playlist feature and autoplay.

## Features

- Play, pause, next, and previous track controls
- Progress bar to see the current track's progress
- Volume control
- Playlist of all available tracks
- Autoplay to the next song when the current one ends

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Use

1. Clone the repository or download the files.
2. Open the `index.html` file in your web browser.
3. The music player will be displayed, and you can start playing music.

## Project Structure

- `index.html`: The main HTML file that contains the structure of the music player.
- `style.css`: The CSS file that styles the music player.
- `script.js`: The JavaScript file that handles the functionality of the music player.
- `music/`: This folder contains the music files.
